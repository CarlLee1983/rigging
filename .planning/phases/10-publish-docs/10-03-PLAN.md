## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-10-05 | Tampering | npm package contents | mitigate | Pre-publish dry-run confirms files manifest matches expected (bin/, lib/, template/ only); .planning/ and packages/ exclusion verified in Task 1 Check 5 |
| T-10-06 | Information Disclosure | template/ published contents | mitigate | build-template.js uses git ls-files + EXCLUDE_PREFIXES to strip .planning/, packages/; Task 1 Check 3+5 verify no leaks before publish |
| T-10-07 | Elevation of Privilege | npm account | accept | Manual npm login required; human reviews publish output before confirming; no automated credentials stored |

</threat_model>
